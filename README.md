# Calculadora com Kivy

Projeto de calculadora simples desenvolvido em Python utilizando Kivy.

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão
- Interface gráfica

## Tecnologias
- Python
- Kivy

## Como executar

Instale o Kivy:

pip install kivy

Depois execute:

python calculadora.py
